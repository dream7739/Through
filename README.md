# Through
### 화이트리스트 기반 송신자 주소 확인 기법을 활용한 이메일 피싱 방지 애플리케이션 

* 한 번의 로그인으로 등록한 모든 이메일 계정의 메일을 확인 가능
* 발﹒수신자 화이트리스트 제공
* 메일 상의 링크 클릭 시 링크 프리뷰 제공
